About Download Files

You can download the maps using the links below.[*1]

We have prepared the maps in different projections as follows.

  • JGD2000 (EPSG-code: 4612; hereafter "JGD")
  • World Mercator (EPSG-code: 54004; hereafter "WM")

The maps have been made by using "Adobe Illustrator" and the file format is [.ai]. For more details about operating environments etc. please see the instruction file [.txt].

[*1] Mac Safari Users: to download the files it may be necessary to hold down 'control' when clicking on the link. This should display appropriate download options.
